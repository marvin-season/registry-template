## 无法科学上网

### GitHub

解决思路：修改 `hosts`。

使用 [ipaddress.com](https://www.ipaddress.com/) 查询 `github.com` 与 `raw.githubusercontent.com` 的 IP 地址。

在服务器 `/etc/hosts` 中新增：

```
140.82.112.4 github.com
185.199.108.133 raw.githubusercontent.com
```
